# Changelog

## 1.0.0

- Initial FastFlow release.
- Added `speckit.fastflow.create` for single-file feature increment specifications.
- Added `speckit.fastflow.implement` for direct implementation from a FastFlow file.
- Added `speckit.fastflow.classify` for routing between tiny change, FastFlow, and full SDD.
- Established Onion Architecture as the default architectural stance for both frontend and backend work.
- Established business-first and functional-first testing as the default testing stance.
